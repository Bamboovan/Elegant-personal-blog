欢迎来到你的新博客！这是一个简约而精致的写作空间，专为追求品质的内容创作者设计。

## 设计哲学

这个博客采用了"编辑式极简主义"的设计理念：

- **温暖米白背景** - 比纯白更柔和，长时间阅读更舒适
- **精致衬线字体** - Newsreader 字体带来优雅的阅读体验
- **充足的留白** - 让内容呼吸，减少视觉疲劳
- **微妙的纸张纹理** - 增添质感而不喧宾夺主

## 开始使用

### 写文章

在 `content/posts` 目录下创建 `.md` 文件，使用以下格式：

```markdown
---
title: "文章标题"
date: "2025-02-07"
excerpt: "文章摘要"
tags: ["标签1", "标签2"]
---

文章内容...
```

### 支持的功能

- **Markdown 语法** - 完整的 Markdown 支持，包括表格、代码块等
- **代码高亮** - 自动语法高亮
- **暗色模式** - 自动跟随系统或手动切换
- **响应式设计** - 适配各种屏幕尺寸
- **RSS 订阅** - 自动生成 RSS 源

## 自定义配置

### 修改个人信息

编辑 `app/layout.tsx` 和 `app/about/page.tsx` 文件，替换为自己的信息。

### 调整样式

设计系统在 `app/globals.css` 中定义，你可以轻松调整：

- 颜色变量
- 字体大小
- 间距
- 其他视觉元素

## 部署

博客已经配置为静态导出，支持部署到任何静态托管服务：

```bash
npm run build
```

生成的 `dist` 目录可以直接部署到 Vercel、Cloudflare Pages、GitHub Pages 等平台。
